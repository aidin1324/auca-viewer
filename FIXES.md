# Исправления проблем с авторизацией и загрузкой данных

## Проблемы, которые были исправлены:

### 1. 🔐 **Ошибка 401 Unauthorized**
**Причина:** Запрос на получение курсов выполнялся до завершения авторизации.

**Решение:**
- Добавлено правильное управление состоянием авторизации
- Запрос курсов теперь выполняется только после успешной авторизации
- Добавлена задержка после авторизации для установки cookies

### 2. ⚛️ **React useEffect dependency array error**
**Причина:** Нестабильные зависимости в массиве useEffect.

**Решение:**
- Исправлен массив зависимостей в useEffect
- Удалены изменяющиеся зависимости, которые могли вызвать проблемы

### 3. 🍪 **Улучшение работы с cookies**
**Решение:**
- Добавлена опция `withCredentials: true` для сохранения сессионных cookies
- Добавлены interceptors для логирования запросов/ответов
- Улучшена обработка ошибок авторизации

## Последовательность выполнения теперь правильная:

```
1. Пользователь вводит данные → 
2. Вызывается authenticate() → 
3. Сервер устанавливает cookie → 
4. Состояние isAuthenticated становится true → 
5. useEffect запускает getAllCourses() → 
6. Запрос выполняется с установленной cookie → 
7. Данные успешно загружаются
```
